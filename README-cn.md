# 没用的软件

### ***其实这个软件什么也不会干...***
<p align="center"><a href="https://github.com/WillamSun/Useless-App/blob/master/README.md">English</a> | 中文</p>

## 用法
按下***开始检测***按钮检测***你的电脑是否开着***。
如果你不开电脑你是打不开这个软件的，所以它***永远显示电脑为开***. 这就是它为啥没用...
## 语言  
即使它没什么用但它还是支持**7种语言**，他们是：
- `英语`
- `简体中文`
- `繁体中文`  
- `法语`
- `德语`
- `俄语`
- `日文`
